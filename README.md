# Zig console Todo

## Commands

```
 list                 ->  List all tasks
 add {name}           ->  Create new task with name
 remove {id}          ->  Remove task by id
 set {id} {status}    -> Update task status(pending | done) by id
 rename {id} {name}   -> Rename task by id
 help                 -> Display available commands
 exit                 -> Exit app
```
